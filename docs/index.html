Objective

Guess the mystery car in 5 attempts using clues from 6 categories.

Categories

Body: Vehicle type (SUV, Sedan, Hatchback, Coupe, Pickup, Wagon, Convertible)

Origin: Country codes (JPN=Japan, GER=Germany, USA=United States, etc.)

Power: Engine type (ICE=Internal Combustion, HYB=Hybrid, EV=Electric)

Market Segment: Broad category (SUV, Family, City, Sports, Eco, Utility)

Color Coding

🟢 Green: Correct match
